ICS-309 Communications Logger

A fast, portable desktop app for capturing radio communications and producing ICS-309 Communications Logs. Runs on macOS, Linux, and Windows — straight from a USB drive, no installation required.

Download Getting Started View on GitHub

Built for the radio operator

Keyboard-driven logging that matches the cadence of a real radio exchange, with the official ICS-309 layout baked in.

⌨️

Keyboard-first logging

Enter moves through the fields in the natural order of a transmission; the time stamps itself on submit.

🔢

Auto message numbers

Message numbers increment automatically per call sign, separately for FROM and TO. Manual entries are never overwritten.

✅

Live spell check

Misspelled words are underlined in the message field, with right-click suggestions — just like a word processor.

📄

Export & print

One-click export to a form-accurate PDF, an Excel workbook, or an FLdigi file — plus direct printing.

💾

Truly portable

Stores everything in a single SQLite file next to the app. Copy it to a flash drive and the whole log travels with you.

🖥️

Cross-platform

One codebase, native builds for macOS (Apple Silicon & Intel), Linux, Windows, and Android.
